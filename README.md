A strategy war game that involves multiple players fighting each other for certain objectives. Set in the world war 2 era using world war 2 historically accurate vehicles, weapons and characters. It will involve historically accurate teams including britain, france, US, Canada, Nazi Germany, Russia etc. Each team will have a character profile which represent the leader of each team. E.g. Aircraft units e.g. B-17 Flying Fortress for US, etc. The game will be played on either mobile phone or computer of each player. Please include background audio of Erika Marschlied - German Victory.
